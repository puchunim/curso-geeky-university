iterator
	- Um objeto que pode ser iterado.
	- Um objeto que retorna um dado, sendo um elemento por vez quando uma função next() é chamada;

	Exemplos:
		lista = [1, 2, 3, 4]
		it = iter(lista)


iterable
	- Um objeto que irá retornar um iterator quando a função iter() for chamada.

	Exemplos:
		nome = 'Pedro'
		lista = [1, 2, 3, 4]

		# Os dois exemplos vão retornar iterators se iter() for usado

iter => Transforma um iterável (iterable), como listas, tuplas, dicionários, strings e etc em objetos iteráveis (iterators).

next => Retorna o próximo elemento de um objeto iterável (iterators).
	Exemplo:
		a = ['pi', 'ca']
		it = iter(a)
		print(next(it)) # pi
		print(next(it)) # ca

	StopIteration será gerado se mais next() forem usados num objeto iterável que já terminou.
