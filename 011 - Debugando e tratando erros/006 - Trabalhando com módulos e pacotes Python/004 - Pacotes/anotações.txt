módulo x pacote =>
	Módulo é um arquivo Python que pode ter diversas funções para utilizarmos.

	Pacote é um diretório contendo uma coleção de módulos.

	OBS: Nas versões antigas do Python, um pacote Python deveria conter dentro dele um arquivo chamado __init__.py

	Nas versões novas isso não é mais obrigatório, mas ainda é usado para manter combatibilidade.
